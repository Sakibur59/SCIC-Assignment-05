## 🌟 Overview

CareerPilot AI is a modern full-stack web application that uses **Google Gemini AI** to analyze resumes, calculate ATS scores, extract skills, and recommend jobs based on user profiles. It provides an intelligent and user-friendly platform for career development.

## ⚙️ Installation

```bash
# Clone repository
git clone https://github.com/Sakibur59/SCIC-Assignment-05.git

# Install frontend
cd frontend
npm install

# Install backend
cd ../backend
npm install
```

Run the project:

```bash
# Backend
npm run dev

# Frontend
npm run dev
```
